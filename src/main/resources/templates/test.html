Password Guidelines

  • Create a strong, unique password (not the one you use for your bank account or email!)
  • Strong passwords are hard to guess, so avoid using "twilio", your personal information like email, repeating characters, or sequential numbers.
  • We recommend using a password manager to generate & manage passwords. If password managers aren't an option, use a passphrase e.g. 0wls are such lovely creatures!
  • Enable two-factor authentication to ensure strong security for your Twilio account
